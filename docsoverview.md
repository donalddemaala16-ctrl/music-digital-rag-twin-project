# Overview

This project represents a Digital Twin of music listening behavior.

## Components
- Data Layer (JSON)
- UI Layer (HTML/CSS/JS)
- Intelligence Layer (Python RAG)

## Flow
User → Input songs → Stored in system → Stats generated → Insights returned
